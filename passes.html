    2. Pick your Pass

  • Rebaze Pass for Builds

    Apache Maven is the most widely adopted software project management and comprehension tool in the Java space. The Maven Pass includes first class assistance for: Developer Onboarding, POM maintenance and Best Practices.

  • Rebaze Pass for Runtimes

    The OSGi Framework is a module system for the Java platform. As an Alliance Member we can provide hands-on services for adopting the technology and troubleshooting. The OSGi Pass includes tooling assistance and migration support.

  • Rebaze Pass for DevOps

    Devops depends on tool availability. To cover deployments of your Java Platform based technology stack in fully automated way, you need a conceptual understanding of the technologies involved. The DevOps Pass includes reviews of your current automation, tool suggestions and extended coaching for Chef and Puppet.

  • Rebaze Pass for Persistence

    Neo4J is a bright example of polyglot persistence. As a consulting partner of Neo Technology we can deliver first class support for adopting this technology. The Persistence Pass includes assistance for migration projects as well as architecture reviews.
