  • Rebaze Pass for Maven

    Apache Maven is a software project management and comprehension tool. Based on the concept of a project object model (POM), Maven can manage a project's build, reporting and documentation from a central piece of information.

    Maven certainly is not the only build system, and it is far from being perfect. But we think it is the most complete system covering the entire build and deployment lifecycle using a declarative approach.

    Contact us!

  • Rebaze Pass for OSGi

    The OSGi Framework is a module system and service platform for the Java programming language that implements a complete and dynamic component model, something that as of 2012 does not exist in the stock Java installment.

    Rebaze is a member at the OSGi Alliance and has deep expert insight into the OSGi specification. Therefore we can provide hands-on services for adopting the technology and troubleshooting.

    Contact us!
